# Amazon Sales SQL Analytics

## Overview

Amazon Sales SQL Analytics is an end-to-end SQL analytics project built on a large-scale e-commerce dataset containing 100,000 transaction records.

The objective of this project is to analyze customer behavior, product performance, sales trends, payment preferences, and operational metrics using SQL. The project demonstrates how raw transactional data can be transformed into meaningful business insights that support data-driven decision making.

This project is designed to showcase practical SQL skills commonly required for Software Engineering, Data Analytics, Business Intelligence, and Database-related roles.

---

## Problem Statement

E-commerce platforms generate large amounts of transactional data every day. Organizations rely on data analysis to understand customer behavior, optimize product strategies, improve operational efficiency, and increase revenue.

This project focuses on solving business-oriented analytical problems using SQL and real-world sales data.
